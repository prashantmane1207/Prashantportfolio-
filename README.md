# Personal Portfolio Website

## Overview

This is my personal portfolio website created using HTML and CSS. The website showcases my profile, skills, education, projects, and contact information in a clean and responsive design.

## Technologies Used

* HTML5
* CSS3

## Features

* Responsive portfolio design
* About Me section
* Skills section
* Education details
* Project showcase
* Contact information
* Clean and user-friendly interface

## Project Structure

```text
Portfolio/
│
├── index.html
├── style.css
├── images/
└── README.md
```

## How to Run

1. Download or clone the repository.
2. Open `index.html` in any web browser.
